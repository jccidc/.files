# .files v0.4.0 Feature Design

Date: 2026-03-20

## Overview

Six features based on community feedback from r/software release post.

## 1. Widget Toggles

Formalize the existing `WIDGET_REGISTRY` in Titlebar.tsx. Add `enabled_widgets` to settings. Settings panel gets a widget management section where you toggle each widget on/off and choose placement (titlebar/footer/off). The registry pattern already half-exists -- just needs settings wiring cleaned up so it persists properly and defaults are sensible.

### Changes
- `settings.rs` / `types.ts`: ensure `titlebar_widgets` and `footer_widgets` arrays persist correctly
- `SettingsPanel.tsx`: clean up widget toggle section, make it the canonical way to manage widgets
- `Titlebar.tsx`: read from settings store, respect enabled/disabled state

## 2. Accessed Column + Date Filter

Add `accessed` timestamp to file metadata. New column in the file list. Sortable with age coloring.

### Changes
- `file_entry.rs`: add `accessed: String` field, populate from `metadata.accessed()`
- `types.ts`: add `accessed` to `FileEntry` interface
- `ExplorerTab.tsx`: add `'accessed'` to `ALL_COLUMNS` and `ColumnId`, wire into `sortEntries()` and `CellValue()`
- `Toolbar.tsx`: add date range filter dropdown (Today, This Week, This Month, This Year, Older)

## 3. Breadcrumb Dropdowns

Each path segment in the breadcrumb bar gets a small chevron. Click opens a dropdown listing sibling folders at that path level.

### Changes
- `Toolbar.tsx`: add chevron button per breadcrumb segment
- New dropdown component (portaled to body like ContextMenu)
- Async `readDir` of parent path, filtered to folders only
- Cache results to avoid repeated reads
- Highlight current folder in the list
- Arrow key navigation

## 4. File Status Tags

Fixed set of ~7 status icons users can assign to files. One tag per file. Stored in settings.

### Tag Types
- Important (exclamation)
- Archive (box)
- Favorite (star)
- Private (lock)
- Done (checkmark)
- In Progress (arrows/spinner)
- Pinned (pin)

### Storage
- `file_tags: Record<string, string>` in AppSettings (normalized path -> tag ID)
- Stored in settings.json, survives restarts

### UI
- New small icon column (~28px) to the left of Name column
- Right-click -> "Tag as..." submenu with icon options
- Clicking same tag again removes it (toggle behavior)
- Batch tagging on multi-select
- Toolbar filter: show by specific tag, "any tagged", or "untagged"

## 5. Draggable Column Headers

All columns can be reordered by dragging their headers. Order persists globally.

### Changes
- `ExplorerTab.tsx`: add pointer-event drag on column headers (same pattern as widget drag)
- `settings.ts` / `types.ts`: add `column_order: string[]` to AppSettings
- Global setting -- all tabs share same column layout
- Column widths already resize; this adds reorder on top

## 6. Archive Support (7z/rar/tar)

Extract and compress archives beyond ZIP. Preview archive contents.

### Extract (context menu)
- "Extract Here" -- extract to current folder
- "Extract to <archive-name>/" -- extract to subfolder
- Formats: .zip, .7z, .rar, .tar, .tar.gz, .tar.bz2

### Compress (context menu)
- "Compress to..." with format picker (ZIP, 7z)
- Select files -> right-click -> choose format -> name the archive

### Preview
- Show archive file listing in preview pane without extracting

### Rust Crates
- `sevenz-rust` for 7z
- `zip` crate (already present) for zip
- `tar` + `flate2` for tar/gz
- `unrar` or shell to unrar.exe for rar

## 7. Date Age Color Legend

The file age coloring (green = today, cyan = this week, gray = older) has no explanation in the UI. Add a small legend.

### Options
- Tooltip on the Modified/Accessed column header showing the color scale
- Small legend row in Settings > Explorer section
- Both

### Recommended
Tooltip on column header hover -- minimal UI footprint, discoverable. Also add a note in Settings.

## Implementation Order

1. Widget toggles (smallest, cleans up existing code)
2. Accessed column (straightforward metadata + column addition)
3. Draggable column headers (pairs with accessed column work)
4. File status tags (new system, medium complexity)
5. Breadcrumb dropdowns (async loading + new UI component)
6. Archive support (new Rust commands + crate dependencies)
7. Date age color legend (small tooltip + settings note)
